Just simply check your variables in Query Monitor result. You must install <a href="https://wordpress.org/plugins/query-monitor/">Query Monitor</a> first.

== Installation ==
1. Download the plugin zip package and extract it.
1. Put the folder named "query-monitor-check-var" under /wp-content/plugins/ directory
1. Go to the plugins page in your Wordpress admin panel and click "Activate"

== Usage ==
1. Using console($var); function, you can check what the variable contains.
1. You can check it on the Query Monitor - Variable Checking.
